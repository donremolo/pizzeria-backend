== RECURSOS DE CATEGORIA
Se puede administrar una lista de categorias de los productos de su pizzeria. Para poder agregar, modificar o eliminar una categoria debe iniciar session como administrador. Para obtener una lista de los productos no es necesario iniciar session.

=== Obtener lista de categorias
La lista de categorias de esta API estan agrupada en paginas que usted puede ir recorriendo. El siguiente es un ejemplo de como solicitar la lista de categorias usando http request. Note que la URL tiene parametros. En primer lugar esta solicitando la primera pagina page=0. Segundo se pide que se incluyan 2 categorias por pagina size=2. Tercero se pide que la pagina se organize en orden alfabetico por nombre de categoria sort=name. Y cuarto que el orden alfabetico debe ser descendente al agregar un sort mas al colocar una coma ,desc.
